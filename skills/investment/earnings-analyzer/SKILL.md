# Earnings Analyzer - 财报分析器

分析上市公司财务报表，提取关键指标和投资洞察。

## Trigger Phrases

- "分析财报"
- "财务报表分析"
- "解读财报"
- "earnings analysis"
- "financial statement"
- "季报/年报分析"

## Key Metrics 关键指标

### Valuation Metrics 估值指标

| Metric | Formula | 说明 |
|--------|---------|------|
| P/E | 股价/EPS | 市盈率 |
| P/B | 股价/每股净资产 | 市净率 |
| P/S | 市值/营收 | 市销率 |
| EV/EBITDA | 企业价值/EBITDA | 企业估值倍数 |
| PEG | P/E/增长率 | 增长调整估值 |

### Profitability Metrics 盈利指标

| Metric | Formula | 说明 |
|--------|---------|------|
| ROE | 净利润/股东权益 | 净资产收益率 |
| ROA | 净利润/总资产 | 总资产收益率 |
| ROIC | NOPAT/投入资本 | 投入资本回报率 |
| 毛利率 | 毛利/营收 | - |
| 净利率 | 净利润/营收 | - |

### Efficiency Metrics 运营效率

| Metric | Formula | 说明 |
|--------|---------|------|
| 资产周转率 | 营收/总资产 | - |
| 存货周转天数 | 365*存货/成本 | - |
| 应收账款周转天数 | 365*应收/营收 | - |
| 应付账款周转天数 | 365*应付/成本 | - |
| 现金转换周期 | 存货+应收-应付 | - |

### Leverage Metrics 杠杆指标

| Metric | Formula | 说明 |
|--------|---------|------|
| 资产负债率 | 负债/资产 | - |
| 利息保障倍数 | EBIT/利息支出 | - |
| 净负债/EBITDA | (有息负债-现金)/EBITDA | - |

## Industry Benchmarks 行业基准

### 科技行业
- 毛利率: 40-60%
- 净利率: 15-25%
- R&D占比: 15-25%

### 消费行业
- 毛利率: 30-50%
- 净利率: 8-15%
- 营销占比: 10-20%

### 金融行业
- ROE: 10-15%
- 不良率: <2%
- 资本充足率: >12%

## Usage

```bash
# Analyze earnings report
bash scripts/llm.sh "分析苹果公司2024年Q4财报，重点分析收入结构变化" --system "你是一位专业的财务分析师"

# Generate financial analysis report
bash scripts/llm.sh "生成腾讯控股的财务分析报告，包含三表分析和关键指标" --json

# Compare companies
bash scripts/llm.sh "对比分析茅台和五粮液的财务指标差异"
```

## Notes

- 数据应来自官方财报或可靠数据源
- 同比/环比分析需注明基数效应
- 行业对比需选择可比公司
- 风险提示必不可少
