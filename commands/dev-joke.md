# /dev-joke - 开发者笑话

开发者笑话、编程幽默和技术梗图。调试时提振士气必备。

## 用法

```
/dev-joke                    # 随机笑话
/dev-joke python             # Python 相关
/dev-joke "debug"            # 调试主题
```

## 笑话类型

| 类型 | 说明 |
|------|------|
| 语言梗 | Python, JavaScript, Java... |
| 职场梗 | 需求变更、上线、加班 |
| 调试梗 | Bug、Stack Overflow、print调试 |
| 架构梗 | 微服务、数据库、部署 |

## 示例输出

```
为什么程序员总是分不清万圣节和圣诞节？

因为 Oct 31 = Dec 25
(八进制31 = 十进制25)
```
