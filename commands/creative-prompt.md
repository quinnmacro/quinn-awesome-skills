# /creative-prompt - 创意提示生成器

生成创意提示，用于写作、设计、头脑风暴和创意构思。

## 用法

```
/creative-prompt <主题>
/creative-prompt "科幻小说"
/creative-prompt "品牌设计" --style professional
```

## 工作流

1. **主题分析** - 理解创作意图
2. **维度拓展** - 多角度发散
3. **提示生成** - 结构化输出

## 输出类型

| 类型 | 说明 |
|------|------|
| 写作 | 故事开头、情节转折、角色设定 |
| 设计 | 视觉风格、配色方案、布局建议 |
| 头脑风暴 | 创意点子、解决方案、新角度 |
| 营销 | 标语、文案、活动创意 |

## 输出格式

```markdown
# 创意提示: [主题]

## 核心概念
## 创意方向
### 方向 1
### 方向 2
### 方向 3
## 参考案例
## 延伸思考
```
