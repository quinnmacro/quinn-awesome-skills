# /code-poet - 代码诗人

将代码转化为诗歌，编写代码主题诗，或用诗意语言解释代码。

## 用法

```
/code-poet <代码片段>
/code-poet "快速排序算法"
/code-poet --style 唐诗
/code-poet --style haiku
```

## 诗歌风格

| 风格 | 说明 |
|------|------|
| 唐诗 | 五言/七言律诗 |
| 宋词 | 词牌格式 |
| Haiku | 日本俳句 |
| Sonnet | 十四行诗 |
| 现代 | 自由诗 |

## 工作流

1. **代码理解** - 分析代码逻辑和意图
2. **意境提取** - 提取核心概念和美感
3. **诗歌创作** - 按选定风格输出

## 示例输出

```
《递归》

函数自调用，
层层深入求解，
终遇基准归。
```
