# /earnings-analyzer - 财报分析器

分析上市公司财务报表，提取关键指标和投资洞察。

## 用法

```
/earnings-analyzer <公司名称>
/earnings-analyzer 苹果 2024Q4
/earnings-analyzer "腾讯控股" --report
```

## 工作流

1. **数据获取** - 获取财务报表数据
2. **三表分析** - 损益表、资产负债表、现金流量表
3. **指标计算** - 估值、盈利、效率、杠杆指标
4. **投资建议** - 综合评价和风险提示

## 分析框架

### 三表分析
- **损益表**: 收入增长、毛利率、费用率
- **资产负债表**: 资产结构、负债水平、ROE/ROA
- **现金流量表**: 经营现金流、自由现金流

### 关键指标

| 类别 | 指标 |
|------|------|
| 估值 | P/E, P/B, P/S, EV/EBITDA |
| 盈利 | ROE, ROA, ROIC, 毛利率 |
| 效率 | 资产周转率、存货周转天数 |
| 杠杆 | 资产负债率、利息保障倍数 |

### 杜邦分析

```
ROE = 净利率 × 资产周转率 × 权益乘数
```
