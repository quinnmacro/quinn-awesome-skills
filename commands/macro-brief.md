# /macro-brief - 宏观经济简报

生成专业的宏观经济简报和市场分析报告。

## 用法

```
/macro-brief                    # 每日简报
/macro-brief weekly             # 周度回顾
/macro-brief "美国通胀分析"      # 专题报告
```

## 工作流

1. **数据收集** - 获取关键经济指标
2. **市场分析** - 资产表现、宏观事件
3. **报告生成** - 结构化输出

## 简报类型

| 类型 | 说明 |
|------|------|
| 每日简报 | 市场概览、宏观要闻、数据日历 |
| 周度回顾 | 资产表现、宏观事件、下周展望 |
| 专题报告 | 深度分析、数据支持、风险因素 |

## 关键指标

- **美国经济**: 非农就业、CPI/PCE、GDP、FOMC
- **中国经济**: PMI、工业增加值、社融/信贷、LPR
- **全球市场**: 美债收益率、美元指数、VIX、油价/金价

## 分析框架

```
增长-通胀矩阵：
        低通胀           高通胀
高增长  | 衰退复苏       | 过热风险
--------+----------------+--------
低增长  | 衰退/滞胀      | 滞胀
```
