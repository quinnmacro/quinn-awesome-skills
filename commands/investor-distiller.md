# /investor-distiller - 投资大师蒸馏器

将投资大师的投资哲学、方法论蒸馏为可执行的分析框架。

## 用法

```
/investor-distiller <投资大师名称>
/investor-distiller 巴菲特
/investor-distiller "Ray Dalio 投资原则"
```

## 工作流

1. **哲学提炼** - 提取核心投资理念
2. **框架生成** - 生成选股清单和分析框架
3. **实践应用** - 案例分析和适用性评估

## 支持的投资大师

### 价值投资
- Warren Buffett - 护城河、安全边际
- Charlie Munger - 多元思维模型
- Benjamin Graham - 安全边际、市场先生

### 宏观与对冲基金
- Ray Dalio - 原则、债务周期
- George Soros - 反身性理论
- Stanley Druckenmiller - 趋势跟随

### 成长与科技
- Peter Lynch - 十倍股
- Bill Ackman - 激进投资
- Cathie Wood - 创新颠覆

## 输出格式

```markdown
# [投资大师名称] 投资框架蒸馏

## 核心理念
## 投资框架
### 选股标准

### 风控原则

## 适用场景
```
